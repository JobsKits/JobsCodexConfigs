## 🔥 <font id=前言>前言</font>

> 这里放置用于 GitHub 大文件规避的压缩拆分与子卷合并脚本。`【MacOS】Codex配置注入替换工具.command` 会在检测到账户配置目录存在拆分子卷时，调用 `【MacOS】🧩子卷➤合而为一源文件.command` 的非交互合并入口。

## 一、脚本清单

- `【MacOS】✂️源文件压缩拆分➤子卷.command`：把大文件拆成 `原文件名@001of003` 这类子卷。
- `【MacOS】🧩子卷➤合而为一源文件.command`：把子卷合并回源文件；同时提供给 Codex 注入脚本使用的非交互入口。
